# Extension Icons

Place your extension icons here:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

You can create these icons using any image editor or online icon generator.

## Quick Icon Creation

1. Create a 128x128 pixel icon with your design
2. Save it as `icon128.png`
3. Resize to 48x48 and save as `icon48.png`
4. Resize to 16x16 and save as `icon16.png`

## Icon Design Tips

- Use a simple, recognizable design
- Ensure icons are readable at small sizes
- Use high contrast colors
- Consider using a microphone or note-taking theme
