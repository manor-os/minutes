# Minutes — Roadmap

## Current (v1.0)
- [x] Live recording with real-time transcription
- [x] Upload pre-recorded audio (MP3, WAV, M4A, WebM, OGG, FLAC, MP4)
- [x] AI-powered summarization, key points, and action items extraction
- [x] Meeting templates (standup, 1-on-1, retrospective, brainstorm, client call, interview)
- [x] Search, sort, and filter meetings
- [x] Tags and favorites
- [x] Meeting sharing via read-only public links
- [x] Bulk delete
- [x] Dark mode (system, dark, light)
- [x] Webhook notifications (Slack, Discord, generic)
- [x] Browser notifications on completion
- [x] Audio playback in meeting detail
- [x] Keyboard shortcuts
- [x] MinIO object storage for audio files
- [x] Dual edition: Community (self-hosted) and Cloud
- [x] Local mode: faster-whisper + Ollama (no API keys needed)
- [x] MCP server for Claude Desktop integration (cloud edition)
- [x] PWA support
- [x] CSS design system with custom properties

## Next (v1.1)
- [ ] Speaker diarization (who said what) via pyannote-audio
- [ ] Calendar integration (Google Calendar, Outlook)
- [ ] Email notifications (SMTP) on meeting completion
- [ ] Meeting title rename from list view
- [ ] Pagination for large meeting lists
- [ ] Audio waveform visualization during playback
- [ ] Export to Google Docs / Notion

## Future (v2.0)
- [ ] Multi-language UI (i18n)
- [ ] Real-time collaborative editing of meeting notes
- [ ] Team workspaces with role-based access
- [ ] Analytics dashboard (meeting frequency, talk time, trends)
- [ ] Browser extension for Google Meet / Zoom / Teams
- [ ] Mobile native apps (iOS, Android)
- [ ] S3 / GCS cloud storage backends
- [ ] Custom LLM model support (bring your own endpoint)
- [ ] Meeting recording scheduling (auto-record calendar events)
- [ ] Transcript search across all meetings (full-text search)
