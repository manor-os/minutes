# Minutes

**AI Meeting Notes — Open Source, Self-Hosted, Privacy-First.**

Minutes records your meetings, transcribes them, generates AI summaries, extracts key points and action items — all running on your own infrastructure.

## Why Minutes?

- **Privacy-first** — Run 100% locally with no data leaving your machine
- **Self-hosted** — Your data, your server. No third-party access
- **Open source** — MIT license, fully customizable
- **Two modes** — Local AI (no API keys) or Cloud API (better accuracy)
- **One command install** — `npx create-manor-minutes`

## How It Works

1. **Record** — Live recording with real-time transcription, or upload a pre-recorded file
2. **Process** — AI transcribes audio, identifies speakers, generates summary
3. **Review** — Read summary, key points, action items. Ask follow-up questions via AI chat
4. **Share** — Export as text/PDF, share read-only links, send to Slack/Discord via webhooks

## Quick Start

```bash
npx create-manor-minutes
```

Or see the [Installation Guide](getting-started/installation) for manual setup.

## Part of Manor AI

Minutes is the first standalone module from [Manor AI](https://manorai.xyz) — an operating system for one-person companies. Meeting notes flow into your Manor knowledge base, and AI agents can reference them to execute tasks.
