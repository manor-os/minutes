# Search

## Quick Search

Type in the search bar on the Meetings page (or press `Cmd+K`). Results appear in real-time as you type (3+ characters).

Searches across:
- Meeting titles
- Full transcripts
- AI summaries

## Search Results

Each result shows:
- Meeting title
- Where the match was found (title / transcript / summary badges)
- Context snippet with the matching text
- Click to open the meeting

## Sort & Filter

| Filter | Options |
|--------|---------|
| **Sort** | Newest, Oldest, Longest, Shortest |
| **Status** | All, Completed, Processing, Failed |
| **Favorites** | Show only starred meetings |
