# Meeting Sharing

Share meeting notes with anyone via read-only public links.

## Create a Share Link

1. Open a meeting
2. Click **Share** in the footer
3. Copy the generated link

The link provides read-only access to: title, summary, key points, action items, and transcript. No login required for viewers.

## Revoke Access

Click **Shared** → **Revoke** to disable the link. Previously shared URLs will return a 404.

## What's Shared

| Included | Not Included |
|----------|-------------|
| Title | Audio file |
| Summary | Internal IDs |
| Key points | Entity/user info |
| Action items | Token cost |
| Transcript | Edit access |
