# Meeting Templates

Templates customize how AI summarizes your meetings. Each template has tailored prompts for summary, key points, and action items.

## Available Templates

| Template | Best For |
|----------|----------|
| **General** | Any meeting (default) |
| **Daily Standup** | Scrum-style updates — yesterday/today/blockers per person |
| **1-on-1** | Manager-report meetings — feedback, goals, concerns |
| **Retrospective** | Sprint retros — what went well / needs improvement / actions |
| **Brainstorm** | Ideation sessions — group ideas by theme |
| **Client Call** | External meetings — requirements, commitments, timeline |
| **Interview** | Candidate evaluation — strengths, weaknesses, next steps |

## How to Use

Select a template in the Recorder before starting a recording or uploading a file. The template is stored with the meeting and used during AI processing.
